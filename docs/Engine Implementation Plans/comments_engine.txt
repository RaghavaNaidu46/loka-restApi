# Strongly Comments Engine — Implementation Plan

This plan details the design and implementation of a robust, highly interactive, and civic-oriented comments system for Loka. The system will support threaded replies, upvoting (likes), filtering comments by citizen stance (Support, Oppose, Neutral), sorting, and visual badge markers (Creator/Official/Stance badges).

## User Review Required

> [!IMPORTANT]
> Database Migration: Since we are running on PostgreSQL, we will write a new Alembic migration to add the new tables and columns (`parent_id` to `comments`, new `comment_likes` table, and new `comment_reports` table).
> 
> UI Styling: Comment lists will be threaded up to 1 level (replies to comments, but replies cannot have replies). This keeps the mobile interface clean and easy to navigate.

---

## Open Questions

> [!NOTE]
> **Stance badges**: Stance badges are computed dynamically based on the commenter's participation status for the issue. If the commenter has supported, they get a "Supported" badge. If they opposed, they get an "Opposed" badge.
> 
> **Official roles**: We will display the commenter's role (e.g. "official", "moderator", etc.) as a badge to elevate authoritative responses.

#### [NEW] [comment_like.py](file:///c:/Users/aravi/OneDrive/Desktop/Loka/Loka/backend/app/models/comment_like.py)
Create a new join table for tracking comment likes:
```python
import uuid
from sqlalchemy import ForeignKey, DateTime, UniqueConstraint, func
from sqlalchemy.orm import Mapped, mapped_column, relationship
from sqlalchemy.dialects.postgresql import UUID
from app.core.database import Base

class CommentLike(Base):
    __tablename__ = "comment_likes"

    id: Mapped[uuid.UUID] = mapped_column(UUID(as_uuid=True), primary_key=True, default=uuid.uuid4)
    citizenId: Mapped[uuid.UUID] = mapped_column(UUID(as_uuid=True), ForeignKey("citizens.id"), nullable=False, name="citizen_id")
    commentId: Mapped[uuid.UUID] = mapped_column(UUID(as_uuid=True), ForeignKey("comments.id", ondelete="CASCADE"), nullable=False, name="comment_id")
    createdAt: Mapped[DateTime] = mapped_column(DateTime(timezone=True), server_default=func.now(), name="created_at")

    __table_args__ = (
        UniqueConstraint("citizen_id", "comment_id", name="uq_citizen_comment_like"),
    )
```

#### [NEW] [comment_report.py](file:///c:/Users/aravi/OneDrive/Desktop/Loka/Loka/backend/app/models/comment_report.py)
Create a new table for comment reports:
```python
import uuid
from sqlalchemy import ForeignKey, DateTime, Text, func
from sqlalchemy.orm import Mapped, mapped_column
from sqlalchemy.dialects.postgresql import UUID
from app.core.database import Base

class CommentReport(Base):
    __tablename__ = "comment_reports"

    id: Mapped[uuid.UUID] = mapped_column(UUID(as_uuid=True), primary_key=True, default=uuid.uuid4)
    citizenId: Mapped[uuid.UUID] = mapped_column(UUID(as_uuid=True), ForeignKey("citizens.id"), nullable=False, name="citizen_id")
    commentId: Mapped[uuid.UUID] = mapped_column(UUID(as_uuid=True), ForeignKey("comments.id", ondelete="CASCADE"), nullable=False, name="comment_id")
    reason: Mapped[str] = mapped_column(Text, nullable=False)
    createdAt: Mapped[DateTime] = mapped_column(DateTime(timezone=True), server_default=func.now(), name="created_at")
```
